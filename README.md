# Notice Board CRUD Application

A full-stack Notice Board application built using Next.js, Prisma ORM, MySQL (TiDB Cloud), and Tailwind CSS.

## Features

- Create Notice
- View Notices
- Edit Notice
- Delete Notice
- Priority Badge
- Category
- Publish Date
- Image Support

## Tech Stack

- Next.js
- Prisma ORM
- TiDB Cloud (MySQL)
- Tailwind CSS

## Installation

```bash
npm install
```

Create a `.env` file

```
DATABASE_URL=your_database_url
```

Run

```bash
npx prisma db push
npm run dev
```

Open

```
http://localhost:3000
```
